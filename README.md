# Portfolio

React + TypeScript + Vite 기반의 포트폴리오 웹사이트와 Hono API 서버

## 프로젝트 구조

- **Frontend**: React + TypeScript + Vite
- **Backend**: Hono + TypeScript
- **Package Manager**: pnpm

## 개발 환경 설정

```bash
# 의존성 설치
pnpm install

# 프론트엔드 개발 서버 실행
pnpm dev

# 백엔드 개발 서버 실행
pnpm dev:server
```

## 빌드

```bash
# 프론트엔드 빌드
pnpm build

# 백엔드 빌드
pnpm build:server
```

## 실행

```bash
# 프론트엔드 미리보기
pnpm preview

# 백엔드 실행
pnpm start:server
```
